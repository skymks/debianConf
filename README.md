## Vim配置

### 快捷键
- `,+cr :noh` 取消搜索高亮
- `,+n` 打开NerdTree
- `,+h` 打开TAGList
- `,+tn` 新建Tab页
- `,+ba` 清空所有Buffer
- `,+j` 向左切换TAB
- `,+k` 向右切换TAB
- `[range]t {address}` 复制指定的行到指定的行下 
- `[range]m {address}` 剪切指定的行到指定的行下
- `[range]y[ank] [x]` 复制指定范围行到寄存器x

### 一些命令
- `:version` 打印VIM的版本信息
